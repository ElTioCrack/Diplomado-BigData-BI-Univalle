# Fuente de datos: Registro de Nacimientos - Ministerio de Salud de Bolivia

## Origen
- **Institución:** Ministerio de Salud y Deportes del Estado Plurinacional de Bolivia
- **Plataforma:** Portal de Datos Abiertos Bolivia
- **Conjunto de datos original:** Cantidad de nacimientos registrados, 2009-2015
- **Enlace del conjunto:** https://datos.gob.bo/dataset/cantidad-de-nacimientos-registrados-2009-20152
- **PDF de respaldo:** `Cantidad de nacimientos registrados, 2009-2015 - Conjuntos de datos - Datos Abiertos Bolivia.pdf` (copia de la página de descarga)

## Descripción del dataset
Registro administrativo de nacidos vivos y nacidos muertos (**mortinatos**) por municipio, departamento y establecimiento de salud. Incluye información sobre atención por personal de salud.
****
## Período de cobertura
2009 - 2015 (7 años)

## Estructura de los archivos
Cada archivo CSV contiene las siguientes columnas:

| Campo | Descripción |
|-------|-------------|
| id | Identificador único de registro |
| cod_dep | Código numérico del departamento |
| departamento | Nombre del departamento |
| cod_mun | Código numérico del municipio |
| municipio | Nombre del municipio |
| nacidos_vivos | Número de nacidos vivos registrados |
| nacidos_vivos_atendidos_por_personal_de_salud | Número de nacidos vivos atendidos por personal de salud |
| nacidos_muertos | Número de nacidos muertos (mortinatos) |
| nacidos_muertos_atendidos_por_personal_de_salud | Número de nacidos muertos atendidos por personal de salud |
| total_general | Suma total de nacimientos (vivos + muertos) |
